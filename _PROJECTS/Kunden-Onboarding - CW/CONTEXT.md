## Projekt-Beschreibung
Prozess fuer neue Kunden: Von der ersten Anfrage bis zum fertigen Setup. Beinhaltet Fragebogen, Wissensdatenbank-Erstellung, und Uebergabe.

## Workflow
1. Kunde fuellt Onboarding-Fragebogen aus
2. Antworten werden in Knowledge Base umgewandelt
3. Knowledge Base wird auf Server deployed
4. Kunde bekommt Zugangsdaten
